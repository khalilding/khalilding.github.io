# 上传说明（重要）

1. 将本压缩包里的所有文件解压后，整体上传到你的仓库根目录（`khalilding.github.io`）。
2. 打开仓库 Settings → Pages：
   - Source 选 **Deploy from a branch**
   - Branch 选 **main**，目录选 **/(root)**，保存
3. 等 1~3 分钟，访问 https://khalilding.github.io 查看网站。
4. 替换：
   - `assets/img/avatar.png` 换成你的头像（保留同名或更新 `_config.yml` 路径）
   - `assets/resume/Your_Name_CV.pdf` 换成你的简历 PDF（同名即可）
   - 修改 `_config.yml` 里的链接（Email、LinkedIn 等）
5. 写博客：在 `_posts` 里新增 `YYYY-MM-DD-标题.md` 文件。
6. 加项目：在 `_projects` 里新增 `英文短名.md` 文件。
